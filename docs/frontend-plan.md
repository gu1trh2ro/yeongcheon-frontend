# Frontend Plan

## 목표

영천시 빈집 관리 및 자율주행 방범 대시보드를 구현한다.

프론트는 로봇, 백엔드, AI Agent, 공공데이터 분석 결과를 사용자가 이해하기 쉬운 관제 화면으로 보여준다.

본 프로젝트는 단순 관리자 페이지가 아니라, “별의 도시 영천”이라는 지역 정체성을 반영한 스마트 안전 대시보드를 목표로 한다.

---

## 프로젝트 한 줄 설명

별의 도시 영천의 빈집 현황, 자율주행 순찰 로봇, AI 분석 결과, 실종자 후보 알림을 한 화면에서 관리하는 공공 안전 대시보드.

---

## 주요 기능

1. 영천시 빈집 지도
2. 빈집 위험도 마커
3. 자율주행 로봇 상태
4. 로봇 카메라 이미지 또는 영상 placeholder
5. 자율주행 순찰 경로
6. 빈집 정비 우선순위
7. 재건축 용도 추천
8. 실종자 후보 알림
9. 최근 이벤트 목록
10. 전체 통계 카드
11. 탭별 상세 화면
12. 공공데이터 기반 분석 결과 표시

---

## 기술 스택

- Vite
- React
- TypeScript
- Tailwind CSS
- Naver Maps API, 추후 연동

---

## 개발 방식

백엔드 API가 완성되기 전까지는 `mockData.ts`를 사용한다.

이후 `dashboardApi.ts`에서 실제 백엔드 API 호출로 교체한다.

초기 구현에서는 실제 네이버 지도, 실제 API 호출, WebSocket을 구현하지 않는다.

먼저 mockData 기반으로 공모전 시연용 화면을 완성한다.
