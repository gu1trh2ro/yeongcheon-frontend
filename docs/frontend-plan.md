# Frontend Plan

## 목표

영천시 빈집 관리 및 자율주행 방범 대시보드를 구현한다.

프론트는 로봇, 백엔드, AI Agent, 공공데이터 분석 결과를 사용자가 이해하기 쉬운 관제 화면으로 보여준다.

## 주요 기능

1. 영천시 빈집 지도
2. 빈집 위험도 마커
3. 자율주행 로봇 상태
4. 로봇 카메라 이미지
5. 빈집 정비 우선순위
6. 재건축 용도 추천
7. 실종자 후보 알림
8. 최근 이벤트 목록
9. 전체 통계 카드

## 기술 스택

- Vite
- React
- TypeScript
- Tailwind CSS
- Naver Maps API

## 개발 방식

백엔드 API가 완성되기 전까지는 mockData.ts를 사용한다.

이후 dashboardApi.ts에서 실제 백엔드 API 호출로 교체한다.

## MVP 우선순위

1. 대시보드 레이아웃
2. 더미 데이터 표시
3. 지도 placeholder
4. 네이버 지도 연동
5. 빈집 마커 표시
6. 실종자 후보 알림 UI
7. 정비 우선순위 UI
8. 백엔드 API 연동
9. polling 기반 갱신
