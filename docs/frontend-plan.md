# Frontend Plan

## 목표

영천시 빈집 관리 및 자율주행 방범 대시보드를 구현한다.

프론트는 로봇, 백엔드, AI Agent, 공공데이터 분석 결과를 사용자가 이해하기 쉬운 관제 화면으로 보여준다.

## 주요 기능

1. 영천시 빈집 지도
2. 빈집 위험도 마커
3. 자율주행 로봇 상태
4. 로봇 카메라 이미지
5. 빈집 정비 우선순위
6. 재건축 용도 추천
7. 실종자 후보 알림
8. 최근 이벤트 목록
9. 전체 통계 카드

## 기술 스택

- Vite
- React
- TypeScript
- Tailwind CSS
- Naver Maps API

## 개발 방식

백엔드 API가 완성되기 전까지는 mockData.ts를 사용한다.

이후 dashboardApi.ts에서 실제 백엔드 API 호출로 교체한다.

## MVP 우선순위

1. 대시보드 레이아웃
2. 더미 데이터 표시
3. 지도 placeholder
4. 네이버 지도 연동
5. 빈집 마커 표시
6. 실종자 후보 알림 UI
7. 정비 우선순위 UI
8. 백엔드 API 연동
9. polling 기반 갱신

## 디자인 컨셉

본 대시보드는 “별의 도시 영천”의 지역 정체성을 반영한 스마트시티 공공안전 관제센터 UI를 목표로 한다.

핵심 컨셉:

- Star City Yeongcheon Smart Safety Dashboard
- 다크 네이비 기반의 관제센터 스타일
- 별자리/밤하늘 모티프를 은은하게 반영
- AI/로봇 상태는 시안·블루 계열로 표현
- 위험 알림은 레드 계열로 명확히 강조
- 주요 포인트에는 소프트 골드 색상을 제한적으로 사용

주의사항:

- 관광 홈페이지처럼 보이지 않게 한다.
- 별 아이콘을 과하게 반복하지 않는다.
- 장난감 같은 UI를 피한다.
- 공공기관 관제 시스템처럼 전문적으로 보이게 한다.
